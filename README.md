موقع React مع حاسبات الموارد والتسريعات + واجهة حساب.

## التشغيل محلياً

```bash
cd web
npm install
npm run dev
```

## النشر

- **تلقائي:** push إلى فرع `ishak` أو `main` → GitHub Actions ينشر الموقع
- **يدوي:** `cd web && npm run deploy`

## تفعيل GitHub Pages (مرة واحدة)

1. [Settings → Pages](https://github.com/Vostro213/SULTAN124/settings/pages)
2. **Branch:** `ishak` — **Folder:** `/ (root)` أو `/docs`
3. بعد push انتظر 2–5 دقائق

> `npm run build` ينسخ الموقع تلقائياً إلى `index.html` + `assets/` + `docs/`
